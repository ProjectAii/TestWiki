## Welcome to This Wiki

This wiki is powered by **Lorengine** — a self-hosted, static wiki engine built for game design documentation and any markdown-based knowledge base.

Everything you see is rendered from plain `.md` files stored in the `pages/` directory. No database. No backend server. No accounts. Just text files and a build step.

## What You Can Do Here

- **Read docs:** Navigate using the tree sidebar on the left, or use full-text search
- **Write docs:** Create and edit `.md` files in the `pages/` directory
- **Organise docs:** Use folder structure and frontmatter fields to build a page hierarchy
- **Search docs:** Full-text search is built in — every word in every page is indexed
- **Ask the AI:** The sidebar panel can draft content, summarise pages, and answer questions using the wiki's content as context
