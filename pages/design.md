## Design Overview

This section documents all gameplay systems, balance decisions, and design philosophy.

Start here if you're new to the design; refer to subsections for detailed specifications.

## Design Philosophy

**Simplicity in Rules, Depth in Interactions:** Core mechanics are easy to learn (move → act → end turn) but create complex scenarios through unit abilities, terrain, and resource constraints.

**Asymmetric Factions:** Each faction plays fundamentally differently. A single "correct" strategy should not exist.

**Narrative-First Design:** Missions tell a story. Win conditions support the narrative, not vice versa.
